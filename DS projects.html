Data Science Projects

This page collects a few of my data science projects, which are through Jupyter Notebooks. Each imbedded section links directly to the notebook via html export, so you can see the full workflow.

Geomagnetic Predictive Model

builds a machine-learning model to predict the Kp geomagnetic index using cleaned and engineered features from raw solar-cycle data. It compares baseline and tuned models (including KNN) to forecast short-term geomagnetic activity and visualize prediction accuracy.

Baby Model Project

This notebook cleans and prepares a birth dataset, then trains machine-learning models (KNN and Random Forest) to predict infant birthweight from maternal and pregnancy-related features. It evaluates model performance and identifies which factors contribute most to accurate birthweight predictions.
