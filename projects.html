Projects

A collection of projects spanning interactive visualizations and machine-learning notebooks. The notebooks are embedded directly via HTML export so you can see the full workflow.

01

Geomagnetic Predictive Model

Machine Learning KNN Physics Jupyter Notebook

Builds a machine-learning model to predict the Kp geomagnetic index using cleaned and engineered features from raw solar-cycle data. Compares baseline and tuned models (including KNN) to forecast short-term geomagnetic activity and visualize prediction accuracy.

02

Infant Birthweight Prediction

Machine Learning Random Forest KNN Jupyter Notebook

Cleans and prepares a birth dataset, then trains KNN and Random Forest models to predict infant birthweight from maternal and pregnancy-related features. Evaluates model performance and identifies which factors contribute most to accurate predictions.

03

LIGO Gravitational Wave Signal Processing

Signal Processing Physics FFT Jupyter Notebook

Processes raw strain data from the LIGO Hanford and Livingston detectors to detect the GW150914 gravitational wave event. Applies bandpass filtering, whitening, and spectrogram analysis to extract the characteristic chirp signal from noisy detector output.

04

Neural Network Visualizer

Interactive Neural Networks XOR Problem

An interactive visualization of a neural network training on the XOR problem. Shows real-time updates of the decision boundary, loss history, and gradient flow as the network learns.
